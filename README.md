# SmartShelter GIS 🐾

AI + GIS tabanlı hayvan barınağı ve sokak hayvanları karar destek prototipi.

## Amaç
Hayvan barınaklarının konum, kapasite, ihtiyaç, veterinerlik ve sahiplendirme süreçlerini daha şeffaf ve veriye dayalı hale getirmek.

## Özellikler
- GIS tabanlı harita
- Barınak pinleri
- Veri tablosu
- Temel dashboard metrikleri
- Açık veri entegrasyonu
